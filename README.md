# Portfolio Website

This is a simple portfolio website structure built with HTML, CSS, and JavaScript.

## Structure

- `index.html`: Main HTML file containing the website structure
- `styles.css`: CSS file for styling the website
- `script.js`: JavaScript file for interactivity (smooth scrolling and form validation)

## How to Use

1. Open `index.html` in your web browser to view the website.
2. Customize the content in `index.html` to match your personal information.
3. Modify `styles.css` to change the appearance.
4. Update `script.js` if you need additional functionality.

## Customization

- Replace placeholder text in `index.html` with your actual information.
- Add your own images by placing them in an `images` folder and updating the image sources.
- Adjust colors, fonts, and layout in `styles.css`.

## Troubleshooting

- If the website doesn't load properly, ensure all files are in the same directory.
- For any issues with the form, check the browser console for JavaScript errors.
